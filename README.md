# Colo

A simple command line tool for converting hex colors to other formats for use in CSS

### Installation

```bash
# Clone the repo
git clone https://github.com/theschmocker/colo

# Install
cd colo && pip install --user .
```

### Usage

```bash
colo bada55

# output

Input:  #bada55
To RGB: rgb(186, 218, 85)
To HSL: hsl(74, 64%, 59%)
    
```
